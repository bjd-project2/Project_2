PROJECT OVERVIEW

Project Steps

  • Get the data of species for each national park across United States.
  • Clean the data.
  • Visualize the number of species for each park, categorized by the type.
  • Create a Leaflet Heatmap that shows the areas that needs to be seen (Rare Species) and to be protected (Endangered species).
